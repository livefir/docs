Lets quickly get started. Assuming you have the latest version of Go installed, you need two files counter.go and count.html.

First we will render a pure html page on the server and then enhance it with Fir's alpinejs plugin to make it reactive and update the counter without page reloads.

1. Start with HTML

  • counter.go
  • count.html
Go the the directory where you have these files and run: 
go run counter.go

Open your browser and go to http://localhost:9867 to see the counter in action.

2. Enhance with Alpine.js

  • counter.go
  • count.html
Go the the directory where you have these files and run: 
go run counter.go

Open your browser and go to http://localhost:9867 to see the counter in action.
